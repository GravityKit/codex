Add a top-level menu page in the 'objects' section.

This function takes a capability which will be used to determine whether or not a page is included in the menu.

The function which is hooked in to handle the output of the page must check that the user has the required capability as well.

Parameters

string$capabilityThe capability required for this menu to be displayed to the user.
string$menu_slugThe slug name to refer to this menu by (should be unique for this menu).
callable$functionThe function to be called to output the content for this page.
string$icon_urlThe url to the icon to be used for this menu.
