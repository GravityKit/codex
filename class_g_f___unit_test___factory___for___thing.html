Creates a new factory, which will create objects of a specific Thing.

Parameters
object$factoryGlobal factory that can be used to create other objects on the system
array$default_generation_definitionsDefines what default values should the properties of the object have. The default values can be generators – an object with next() method. There are some default generators: WP_UnitTest_Generator_Sequence, WP_UnitTest_Generator_Locale_Name, WP_UnitTest_Factory_Callback_After_Create.
