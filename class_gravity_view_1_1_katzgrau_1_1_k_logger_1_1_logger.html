Detailed Description

Finally, a light, permissions-checking logging class.

Originally written for use with wpSearch

Usage: $log = new GravityView('/var/log/', GravityView::INFO); $log->info('Returned a million search results'); //Prints to the log file $log->error('Oh dear.'); //Prints to the log file $log->debug('x = 5'); //Prints nothing due to current severity threshhold
