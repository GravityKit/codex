json-stable-stringify

deterministic version of JSON.stringify() so you can get a consistent hash from stringified results

You can also pass in a custom comparison function.

example

1 var stringify = require('json-stable-stringify');
2 var obj = { c: 8, b: [{z:6,y:5,x:4},7], a: 3 };
3 console.log(stringify(obj));

output:

1 {"a":3,"b":[{"x":4,"y":5,"z":6},7],"c":8}

methods

1 var stringify = require('json-stable-stringify')

var str = stringify(obj, opts)

Return a deterministic stringified string str from the object obj.

options

cmp

If opts is given, you can supply an opts.cmp to have a custom comparison function for object keys. Your function opts.cmp is called with these parameters:

1 opts.cmp({ key: akey, value: avalue }, { key: bkey, value: bvalue })

For example, to sort on the object key names in reverse order you could write:

1 var stringify = require('json-stable-stringify');
2 
3 var obj = { c: 8, b: [{z:6,y:5,x:4},7], a: 3 };
4 var s = stringify(obj, function (a, b) {
5  return a.key < b.key ? 1 : -1;
6 });
7 console.log(s);

which results in the output string:

1 {"c":8,"b":[{"z":6,"y":5,"x":4},7],"a":3}

Or if you wanted to sort on the object values in reverse order, you could write:

1 var stringify = require('json-stable-stringify');
2 
3 var obj = { d: 6, c: 5, b: [{z:3,y:2,x:1},9], a: 10 };
4 var s = stringify(obj, function (a, b) {
5  return a.value < b.value ? 1 : -1;
6 });
7 console.log(s);

which outputs:

1 {"d":6,"c":5,"b":[{"z":3,"y":2,"x":1},9],"a":10}

space

If you specify opts.space, it will indent the output for pretty-printing. Valid values are strings (e.g. {space: \t}) or a number of spaces ({space: 3}).

For example:

var obj = { b: 1, a: { foo: 'bar', and: [1, 2, 3] } };
var s = stringify(obj, { space: ' ' });
console.log(s);

which outputs:

1 {
2  "a": {
3  "and": [
4  1,
5  2,
6  3
7  ],
8  "foo": "bar"
9  },
10  "b": 1
11 }

replacer

The replacer parameter is a function opts.replacer(key, value) that behaves the same as the replacer from the core JSON object.

install

With npm do:

1 npm install json-stable-stringify
