get_column_headers (   $screen)

Get the column headers for a screen.

Since
2.7.0

array $column_headers

Parameters
string | WP_Screen$screenThe screen you want the headers for
Returns
array Containing the headers in the format id => UI String

Filter the column headers for a list table on a specific screen.

The dynamic portion of the hook name, $screen->id, refers to the ID of a specific screen. For example, the screen ID for the Posts list table is edit-post, so the filter for that screen would be manage_edit-post_columns.

3.0.0
