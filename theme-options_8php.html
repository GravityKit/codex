twentyeleven_option_page_capability (   $capability)

Change the capability required to save the 'twentyeleven_options' options group.

See also
twentyeleven_theme_options_init() First parameter to register_setting() is the name of the options group.
twentyeleven_theme_options_add_page() The edit_theme_options capability is used for viewing the page.

By default, the options groups for all registered settings require the manage_options capability. This filter is required to change our theme options page to edit_theme_options instead. By default, only administrators have either of these capabilities, but the desire here is to allow for finer-grained control for roles and users.

Parameters
string$capabilityThe capability used for the page, which is manage_options by default.
Returns
string The capability to actually use.
